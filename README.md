# Their Eyes Were Watching God - Study Cards

A flashcard study app for Zora Neale Hurston's *Their Eyes Were Watching God* with 91 questions covering Chapters 1-20.

**Live App:** https://tewwg-study-cards-production.up.railway.app

## Features

- 91 verified flashcards with answers sourced directly from the novel
- Clean, minimal off-white aesthetic
- Flip cards to reveal answers
- Shuffle functionality
- Export to CSV/spreadsheet
- Mobile responsive with swipe navigation
- Keyboard navigation (arrow keys, space/enter to flip)

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide React icons

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deployment

Deployed on [Railway](https://railway.app).
